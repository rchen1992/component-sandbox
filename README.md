-   React
-   TypeScript
-   Storybook
-   Styled Components
-   Jest

# Usage

Setup

```bash
npm install
npm run storybook
```

Build design system for publishing to private npm or github instance:

```bash
npm run build
```

Build storybook for static file hosting:

```bash
npm run build-storybook
```

Run tests

```bash
npm test
```
